Os comandos estudados foram:
git status (Comando para visualizar os arquivos incorporados ao git)
git add -A (Comando para adicionar arquivos novos à tela do git)
git commit -m (Comando para comitar, ou seja, indexar os arquivos do git local ao git web)
git branch
git diff (MOstra extamente o que foi alterado e onde foi alterado)
branch (São versões diferentes do sistema construído. Facilita a manipulalção, por exemplo, de vários usuários/programadores/equipes)
